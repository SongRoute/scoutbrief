# 실행 관측 로그 — draft 오류 클래스 기록

실행별 관측을 기록한다 (ARCHITECTURE.md 부록 제언 이행). §5는 숫자의 실재만
보므로 아래 클래스는 전부 자동 검증을 통과한 상태에서 사람이 관측한 것이다.

오류 클래스 표기:
- **bullpen-a** — vsL xwOBA 값을 직접 해석하며 방향 반전 (높은 값=까다로움 류)
- **bullpen-b** — 순위 의미 자체 반전 (threat_score 상위 → "적극 공략" 류)
- **bvp-반전** — bvp_xwoba 귀속 반전 (이정후가 기록한 생산성을 투수 위협으로 서술)
- **matchup-반복** — 통산 ab_total/hits_total이 구종별 행마다 반복 (표본 귀속 오류)
- **날조-통과** — 원천에 없는 수치가 §5 PASS (화이트리스트·corpus 우연 일치 경유)
- **원자료-전사** — 집계 대신 투구 단위 원자료를 전량 나열 (전부 실재값이라 §5 PASS,
  리포트로서 사용 불가. 2026-07-10 gpt-5.4-mini에서 최초 관측: feltner 15만 자)
- **스플릿-해석** — 주자 스플릿을 usage_pct 비교로만 서술하라는 지시를 어기고
  효과·의도·전략 해석을 부여 (P3 지시문 위반)
- **지시문-혼잡** — 지시 추가가 기존 지시의 이행을 밀어냄 (지시 총량 증가로 일부
  지시가 산출물에서 탈락. 2026-07-10 P5 배포본 count_group 표 축소로 명명,
  5.4-mini #5의 주자 스플릿 서술 누락에 소급 적용)
- 신규 클래스는 비고에 서술하고 다음 행부터 명명한다.

방향 반전 합계 = bullpen-a + bullpen-b + bvp-반전 (문장 단위 실측 건수).
